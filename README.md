# Wireshark-Network-Traffic-Analysis

## Objective
To analyze network traffic using Wireshark and understand common protocols, packet behavior, and suspicious network activity.

---

## Tools Used
- Wireshark
- Windows 11
- Wi-Fi Network Interface

---

## Topics Covered
- HTTP Traffic Analysis
- DNS Request and Response Analysis
- Source and Destination IP Identification
- TCP Three-Way Handshake (SYN, SYN-ACK, ACK)
- TCP Retransmission Detection
- Suspicious Packet Analysis
- Exporting Packet Capture Files (.pcap)

---

## Activities Performed
- Captured live network traffic using Wireshark.
- Applied protocol-based filters such as HTTP, DNS, and TCP.
- Analyzed TCP handshake packets and connection establishment.
- Observed DNS queries and responses.
- Identified source and destination IP addresses from captured packets.
- Investigated TCP retransmission packets for suspicious or unstable traffic behavior.
- Exported packet capture files for future analysis.
